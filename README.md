## 2. Install Laravel
- For laravel packages and dependencies.
```bash
Composer install
```
- Copy `.env.example` to `.env` and change app url, app api url and database info.

- For generate app key.
```bash
php artisan key:generate
``` 

- For generate database
```bash
php artisan migrate 
``` 
```bash
php artisan db:seed 
```
## 3.Run The Project

- To run in browser
```bash
php artisan serve
```
